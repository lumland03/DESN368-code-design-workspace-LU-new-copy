## Design Direction

For my portfolio, I pursued an aesthetic that I have no name for. I liked the topographic background, and the dark teal colors.

| Element | Choice |
|---------|--------|
| Header Font | Josefin Sans |
| Body Font | Lexend |
| Primary Color | `#003147` |
| Accent Color | `#315A5E` |
| Background | `#00111A` |
