Why Use a Reset

The initial problem that CSS resets solved was that no 2 browsers had the same CSS defaults. This made it very difficult to create a webpage that looked identical between multiple browsers. There were tiny differences in each browser, such as slightly different margins or padding, text that was underlined or italicized, etc. Having a reset equalizes all of the differences on different browsers so that each browser is a blank page for the CSS rules to be applied to, making them the same across almost every browser.

Why Use Josh's Over Erics

Josh considers Eric's reset to be outdated, and needed modernized. The main issue that resetting fixed isn't all that common or egregious in modern browsers. Josh's also was more suited for differnt styles and aesthetics and was unopinionated with its reset.

How Are They Used

Resets and Normalize are both used by linking them as stylesheets. Similar to a normal CSS stylesheet. They are applied before the styles file to take priority for the reset or the normalization of the browsers imbedded defaults.

What is a CSS Boilerplate

A CSS boiler plate is similar to a reset or normalize, but it isn't a separate file. The boilerplate is placed at the top of your main stylesheet and sets a base ruleset/setting for the rest of the file, such as using "*" to select all elements and set the margins and padding to 0.

Why Use a Boilerplate

Boilerplates are great at saving time, and unifying elements across pages. They are easy to paste into a CSS file to set the base rules for the page, so you don't have to individually change them every time you make a new page. They also help keep your rules consistent between pages you create for a site.

How is Normalize Different Than Reset

Normalize is different than reset because instead of resetting every default browser element, it only affects some, and leaves some defaults. This way, the page is uniform, but it also saves a bit of time by keeping some default rules you plan on using. Reset completely starts you from a blank slate, while normalize is like a preset to work from.
